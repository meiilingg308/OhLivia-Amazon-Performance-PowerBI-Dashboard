# Amazon-Performance-PowerBI-Dashboard
This is a self-built Power BI project with branđ name Oh'Livia developed from Amazon Australia maketplace raw data to analyze sales performance, advertising efficiency, customer shopping behavior, and LTV trends. The report covers executive KPIs, promotion performance, product conversion, customer purchase patterns, and cohort analysis to generate actionable e-commerce business insights.
# Dashboard Outline
The report is structured into several key sections:

--> Executive Overview: summary of core KPIs such as Sales, Orders, Units Sold, AOV, and Advertising performance.

--> Sales Analysis: tracking sales trends over time, campaign periods, and peak shopping events such as Black Friday.

--> Product Analysis: monitoring product-level performance to identify conversion opportunities.

--> Advertising Analysis: understanding campaign performance, what kind of campaign have a good ROAS, tracking impression, cicks, CVR, monitoring CPC, TACos. 

--> Cohort / LTV Analysis: evaluating how customer lifetime value changes by cohort and identifying retention or repurchase drivers.
# Key Analysis Focus
Through this dashboard, I explored not only what happened but also why it happened. Some of the analysis highlights include identifying Black Friday traffic peaks, observing higher order values during Cyber Monday, detecting advance gift-shopping behavior before major holidays, and analyzing bulk-purchase patterns and LTV fluctuations over time.

This project reflects my ability to transform raw marketplace data into a business-facing dashboard with actionable insights, combining data modeling, DAX, and storytelling to support decision-making in Amazon e-commerce operations.
